AssemblyArtist

class compas_assembly.rhino.AssemblyArtist(assembly, layer=None)

Bases: compas_rhino.artists.NetworkArtist

An artist for visualisation of assemblies in Rhino.

Parameters

  • assembly (compas_assembly.datastructures.Assembly) – The assembly data structure.

  • layer (str, optional) – The base layer for drawing. Default is None, which means drawing in the current layer.

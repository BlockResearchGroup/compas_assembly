AssemblyPlotter

class compas_assembly.plotter.AssemblyPlotter(assembly, **kwargs)

Bases: compas.plotters.plotter.Plotter

An AssemblyPlotter combines the functionality of a NetworkPlotter and a MeshPlotter and uses the same set of axes for all drawing output.

Parameters

assembly (Assembly) – The assembly data structure.

Notes

For all other relevant parameters, see Plotter.

Examples

plotter = AssemblyPlotter(assembly, tight=True, figsize=(12, 8))

plotter.draw_vertices()
plotter.draw_blocks()

plotter.show()

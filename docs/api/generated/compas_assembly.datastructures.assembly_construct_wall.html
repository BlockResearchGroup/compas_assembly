COMPAS Assembly 0.1.7

assembly_construct_wall

compas_assembly.datastructures.assembly_construct_wall(number_of_even_bricks, number_of_courses, width, height, depth, gap)

Construct an assembly from parameters describing a wall.

Parameters

  • number_of_even_bricks (int) – The number of bricks on the even rows.

  • number_of_courses (int) – The number of course rows in the wall.

  • width (float) – The width of the base brick.

  • height (float) – The height of the base brick.

  • depth (float) – The depth of the base brick.

  • gap (float) – The horizontal gap between the bricks.

Returns

Assembly – An assembly data structure describing the wall.

Examples

assembly = assembly_construct_wall(5, 7, 2.0, 0.5, 1.0, 0.1)

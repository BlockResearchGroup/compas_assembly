compas_assembly.datastructures

This package defines various data structures for the representation of discrete/distinct-element assemblies. The individual elements of an assembly are represented by customised versions of the COMPAS mesh data structure (compas.datastructures.Mesh). The relationships between the elements in the assembly are represented by customised versions of the COMPAS network data structure (compas.datastructures.Network).
